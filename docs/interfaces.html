The Golem Programming Language Reference

Interfaces

Interfaces define the various intrinsic capabilities of Golemâ€™s standard types.

Hashable

A value is hashable if it can be a key in a Dict or Set.

Str, Int, Float, Bool, and Tuple are hashable.

Comparable

A value is comparable if it supports the comparison operators >, >=, <, <=, <=>.

Str, Int, Float, and Bool are comparable.

Indexable

A value is indexable if it supports the index operator a[x].

Str, List, Range, Tuple and Dict are indexable.

Lenable

A value is lenabale if it has a length. The builtin function len() returns the length of a lenable value.

Str, List, Range, Tuple, Dict and Set are lenable.

Sliceable

A value is sliceable if it supports the slice operators a[x:y], a[x:], a[:y].

Str and List are sliceable.

Iterable

A value is iterable if its entries can be iterated over. Iterable values can be the subject of a for loop. The builtin function iter() returns a Struct that can be used to iterate over the entries in an iterable value.

Str, List, Range, Dict and Set are iterable.
