Dict

A Dict is an associative array, in which the keys are all hashable.

Valid operators for Dict are:

  • The equality operators ==, !=
  • The index operator a[x]

Dict have a len() and are iterable.

Each iterated element in a Dict is a 2-Tuple containing a key-value pair.

Dict has the following fields:

  • addAll
  • clear
  • contains
  • isEmpty
  • remove

addAll

addAll adds all of the values in the given Iterable to the dict, and returns the modified dict. Each iterated element must be a 2-Tuple containing a key-value pair.
