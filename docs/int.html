Int

Int is the set of all signed 64-bit integers (-9223372036854775808 to 9223372036854775807).

Valid operators for Int are:

  • The equality operators ==, !=
  • The comparison operators >, >=, <, <=, <=>
  • The arithmetic operators +, -, *, /
  • The integer arithmetic operators |, ^, %, &, <<, >>
  • The unary integer complement operator ~
  • The postfix operators ++, --

When applying an arithmetic operator +, -, *, /to an Int, if the other operand is a Float, then the result will be a Float, otherwise the result will be an Int.

Ints are hashable

Int has no fields.
