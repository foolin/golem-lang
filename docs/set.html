Set

A Set is an un-ordered collection of unique, hashable values.

Valid operators for Set are:

  • The equality operators ==, !=

Sets are lenable and iterable.

A Set has the following fields:

  • add
  • addAll
  • clear
  • contains
  • containsAll
  • containsAny
  • copy
  • isEmpty
  • remove
