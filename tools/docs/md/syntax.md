# The Golem Programming Language Reference

## Syntax

### Operator precedence

Golem's operators have the following precedence, from low to high:

| Category       | Operators     |
| -------------  | ------------- |
| or             | <code>&#124;&#124;</code>  |
| and            | `&&`  |
| comparative    | `==`, `!=`, `>`, `>=`, `<`, `<=`, `<=>` |
| additive       | `+`, `-`, <code>&#124;</code>, `^` |
| multiplicative | `*`, `/`, `%`, `&`, `<<`, `>>` |
| unary          | `-`, `!`, `~` |
| postfix        | `++`, `--`  |
